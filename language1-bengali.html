Reading Comprehension Strategy ★★★

Prose Passage — Kivabe korbo

  • Age passage er title o first para scan koro
  • Questions age poro — ki khujte hobe jano
  • Answer gulo usually sequence e thake
  • Vocabulary question e context clue use koro
  • Inferred meaning — direct statement noy, bujhe ber korte hobe

Poetry Comprehension

  • Figurative language: metaphor, simile, alliteration
  • Poet er mood / tone bujhte hobe
  • Central theme ki — main message
  • Imagery — kemon picture create korche
  • Rhyme scheme note koro (ABAB, AABB etc.)

Language Pedagogy ★★★

Exam e 8–10 questions sure

Chomsky er Language Theory

  • LAD — Language Acquisition Device — brain e built-in
  • Shishu janmogate language learn korte programme
  • Universal Grammar — sob language e common structure
  • Nativist Theory — language innate (born with it)
  • Surface structure vs Deep structure of sentences
💡Chomsky bolen — shishu ke sikhate hoy na, nijei shikhey nay — LAD er jonno

Krashen er 5 Hypotheses

  • 1. Acquisition-Learning Hypothesis — Acquisition (natural) vs Learning (conscious)
  • 2. Monitor Hypothesis — Conscious rules correct kore
  • 3. Natural Order Hypothesis — Grammar ek natural order e learn hoy
  • 4. Input Hypothesis (i+1) — Slightly above current level input best
  • 5. Affective Filter Hypothesis — Low anxiety = better language learning

Language Skills — 4 Types

  • Listening (শোনা) — Receptive skill — passive
  • Speaking (বলা) — Productive skill — active
  • Reading (পড়া) — Receptive skill — passive
  • Writing (লেখা) — Productive skill — active
💡Natural order: Listening → Speaking → Reading → Writing

Language Teaching Methods

  • Grammar-Translation Method — Rules mene pore anuvad kora (old method)
  • Direct Method — Target language te direct teach, no translation
  • Audio-Lingual Method — Drilling o repetition — habit formation
  • Communicative Approach (CLT) — Real communication focus — best
  • Task-Based Learning — Real-life tasks diye language shikha
